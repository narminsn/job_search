DropzoneJS plugin for Bootstrap 4 - Advanced Components with daemonite material UI (WIP)

by djibe.

(thx to BS4 + Daemonite Material Design)

Dropzone is an easy to use drag'n'drop library. It supports image previews and shows nice progress bars,
a creation of enyo.

If you see this page unstyled, press Ctrl + F5 to reload it.

Setup

  1. Add <link rel="stylesheet" href="https://cdn.rawgit.com/enyo/dropzone/master/dist/dropzone.css"> before your bootstrap css.
  2. Add dropzone.js to your page AFTER bootstrap.min.js
    <script src="https://cdn.rawgit.com/enyo/dropzone/master/dist/dropzone.js"></script>
  3. DropzoneJS won't do any server-side file processing, you still have to do it on your own.
  4. Read the full documentation here : http://www.dropzonejs.com/#installation
